## 8. Accessibility Annotations

### Всех экранах:

```yaml
Touch Targets:
  - Минимальный размер: 44x44 px
  - Отступы между кликабельными элементами: 8px

Font Sizes:
  - Body: 16px (минимум)
  - Small: 14px
  - Large: 18-24px

Contrast:
  - Text: минимум 4.5:1
  - Large text (>18px): минимум 3:1
  - Interactive elements: минимум 3:1

Focus States:
  - Все интерактивные элементы имеют visible focus state
  - Keyboard navigation по Tab

ARIA Labels:
  - Все иконки без текста имеют aria-label
  - Images имеют alt text
  - Buttons имеют descriptive labels
```
