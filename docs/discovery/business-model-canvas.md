## 2. Unit Economics (Детальный расчёт)

### Assumptions (Базовые допущения)

```yaml
User Acquisition:
  - CAC (средний): 80₽
  - Breakdown:
    - Telegram Ads: 100₽/пользователь (CPA)
    - Блогеры: 50₽/пользователь (партнёрка)
    - Referral: 20₽/пользователь (бонусы)
    - Organic: 0₽/пользователь (15% трафика)

Retention:
  - D1: 70%
  - D7: 40%
  - D30: 25%
  - M3: 15%
  - M6: 10%
  - M12: 8%

Conversion:
  - Free → Premium: 5% (через 2-4 недели использования)
  - Churn (месячный): 10%

ARPU Breakdown (Year 1 average):
  - Подписки: 60₽/месяц
  - Партнёрка: 30₽/месяц
  - Реклама: 10₽/месяц
  - Итого: 100₽/месяц
```

### LTV Calculation (Lifetime Value)

```yaml
Формула:
  LTV = ARPU × Average Lifetime (months)

Average Lifetime:
  = 1 / Monthly Churn Rate
  = 1 / 0.10
  = 10 месяцев (при churn 10%)

LTV:
  = 100₽/месяц × 10 месяцев
  = 1,000₽

С учётом viral growth (referral):
  Viral Coefficient (k) = 0.2 (каждый пользователь приводит 0.2 друга)
  Adjusted LTV = 1,000₽ × (1 + 0.2) = 1,200₽
```

### CAC Payback Period

```yaml
Формула:
  Payback = CAC / ARPU

Расчёт:
  = 80₽ / 100₽ per month
  = 0.8 месяца
  ≈ 24 дня

Вывод: Окупаемость пользователя через 1 месяц (отлично!)
```

### LTV/CAC Ratio

```yaml
Расчёт:
  = 1,200₽ / 80₽
  = 15x

Benchmark:
  - <1: Проблема (не окупаемся)
  - 1-3: Норма (SaaS)
  - >3: Отлично
  - >10: Exceptional

Вывод: 15x — отличная unit-экономика!
```

### Contribution Margin (на пользователя)

```yaml
Revenue per user (годовой):
  = 100₽/месяц × 10 месяцев (lifetime)
  = 1,000₽

Variable Costs per user (годовой):
  = 15₽/месяц × 10 месяцев
  = 150₽

Contribution Margin:
  = 1,000₽ - 80₽ (CAC) - 150₽ (variable)
  = 770₽

Contribution Margin %:
  = 770₽ / 1,000₽
  = 77%

Вывод: 77% маржа — очень хорошо для SaaS!
```

## 4. Break-Even Analysis

### Break-Even Point (BEP)

```yaml
Формула:
  BEP (users) = Fixed Costs / (ARPU - Variable Cost per User)

Месячный расчёт:
  Fixed Costs = 1,350,000₽/месяц
  ARPU = 100₽/месяц
  Variable Cost = 15₽/месяц

  BEP = 1,350,000₽ / (100₽ - 15₽)
      = 1,350,000₽ / 85₽
      = 15,882 активных пользователей

С учётом маркетинга (800K₽/мес):
  BEP = (1,350,000₽ + 800,000₽) / 85₽
      = 25,294 активных пользователей

Вывод: Break-even при ~30,000 платящих пользователей
       Это примерно 600,000 WAU при 5% конверсии
       Прогноз: достижимо в Q3-Q4 2027 (Year 2)
```
